Introduction

Reminder, beyond an image slideshow tool, has the following unique features:

  1. can customize how the image is shown
  2. automatically search and download images(*)
  3. can embed the image with customized phrases

*: ‘api_key’ and ‘cx’ of google custom search is required

Installation

Download and extract the source and copy it into the desired directory

Demo usage

  1. ./main.py
  2. ./main.py config_image.ini
  3. ./main.py config_phrase.ini

In 1., how various image can be searched/downloaded/slide-showed is demonstrated.
In 2., how the image will be shown by customized setting is demonstrated.
In 3., how the image can be embedded with phrases is demonstrated.

Motivation

We are creatures who can be mind-occupied by image and phrases. Hopefully, Reminder helps you determine how your precious memory space will be occupied.
