Introduction | 介紹

Reminder, beyond an image slideshow tool, has the following unique features:
Reminder不僅僅是一個影像幻燈片播放工具，它具有下列獨特的功能：

  1. can customize how the image is shown | 可客製化影像的呈現方式
  2. automatically search and download images | 自動搜尋並下載影像
  3. can embed the image with customized phrases | 可嵌入客製化的語句於影像中

*: for search function, api_key and cx of google custom search is required | 搜尋功能需要指定可以從google custom search中取得的api_key及cx

System requirement | 系統需求

python2.7

Installation | 安裝

  1. Install tkinter | 安裝tkinter
    linux apt-get install python-tk

  2. Install imageTk/PIL | 安裝 imageTk/PIL
    linux apt-get install python-imaging-tk
    osx pip install Pillow

  3. Download and extract the source and copy it into the desired directory.
    And have fun!
    下載原始程式檔案，解開壓縮，並且複製到您想安裝的目的資料夾。
    然後開始享受它帶來的樂趣吧！

Demo usage | 示範用法

  1. ./main.py config.ini
  2. ./main.py config_image.ini
  3. ./main.py config_phrase.ini

which demonstrate the following:
這展示了下面的功能：

  1. image search/download/slide-show functionality | 影像搜尋/下載/幻燈片播放
  2. show image by customized setting | 客製化影像呈現的方式
  3. embed image with phrases | 將語句嵌入至影像

Motivation | 動機

We are creatures who can be mind-occupied by image and phrases.
Hopefully, Reminder helps you determine how your precious memory space will be occupied.
我們人類，是一種易於被“目所見、耳所聞”的形象佔據思考的生物。
但願Reminder能夠幫助我們自行決定那些珍貴的思維空間，將如何被填滿。
